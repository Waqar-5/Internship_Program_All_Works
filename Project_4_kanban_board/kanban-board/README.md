# 📌 Kanban Board - React Project

A simple and responsive Kanban Board application built using React.  
This project helps manage tasks visually by moving them between different stages using drag & drop functionality.

---

## 🚀 Features

- ➕ Add new tasks with title and priority
- 🗑️ Delete tasks from any column
- 🟡 Drag & drop tasks between columns (To Do → Doing → Done)
- 💾 Data persistence using LocalStorage
- 📱 Fully responsive UI design
- 🎯 Priority levels (High / Medium / Low)
- ⚡ Clean and simple user interface

---

## ⚙️ How It Works

### ➤ Add Task
- Click on **Add Task**
- Enter task title and select priority
- Task will be added to **To Do column**

---

### ➤ Drag & Drop
- Drag task card
- Drop it into another column:
  - To Do
  - Doing
  - Done

---

### ➤ Delete Task
- Click delete button on task card
- Task will be removed instantly

---

### ➤ Data Persistence
- All tasks are saved in browser using LocalStorage
- Data remains even after refresh
