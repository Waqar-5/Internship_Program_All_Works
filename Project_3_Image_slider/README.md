## ⚙️ How It Works

### 1. Slide System
Only one image is shown at a time using the `.active` class. JavaScript dynamically changes the active slide.

---

### 2. Navigation Logic
- Next button increases index  
- Previous button decreases index  
- Looping ensures infinite navigation (last → first)

---

## 🔘 4. Dot Navigation

Each dot represents a slide:

- Click dot → jump to that slide  
- Active dot updates dynamically with current slide  

This improves user experience and quick navigation.

---

## ⏸️ 5. Pause on Hover

When mouse enters slider:

- Auto-play stops (user can interact freely)

When mouse leaves:

- Auto-play resumes automatically  

This prevents unwanted slide changes during interaction.
